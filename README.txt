# crypto

ENCODE AND DECODE MESSAGES USING A ONE TIME PAD ENCODING SCHEME-

encode messages using main method input in Crypt (msg) and scanner input (key) in Crypt class,

decode messages using main input of Decrypt (msg) and scanner input (key) in Decrypt class (utilized Converter class to move back to ascii).

more info on how the one time pad works HERE: https://www.khanacademy.org/computing/computer-science/cryptography/ciphers/a/xor-and-the-one-time-pad
